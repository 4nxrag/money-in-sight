
## Project info

This is an Expense tracker that helps you track your income and expenses and show them via a pie chart and keep a record of it every month it feels very interactive and responsive.
You can add your income and expenses using the menu and see them the data is stored in local storage.

**Use your preferred IDE**

Follow these steps:

```sh
# Step 1: Clone the repository using the project's Git URL.
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory.
cd <YOUR_PROJECT_NAME>

# Step 3: Install the necessary dependencies.
npm i

# Step 4: Start the development server with auto-reloading and an instant preview.
npm run dev
```

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
